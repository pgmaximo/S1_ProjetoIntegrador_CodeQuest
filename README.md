# CodeQuest

Projeto Integrador desenvolvido no 1º semestre do curso de Ciência da Computação no Instituto Mauá de Tecnologia.

## Sobre o projeto

O **CodeQuest** é um projeto acadêmico criado para aplicar conceitos iniciais de programação, lógica computacional e desenvolvimento de soluções digitais. A proposta do projeto é reunir aprendizado técnico e prática em equipe em uma solução interativa relacionada ao universo da programação.

Este repositório registra a evolução do projeto, os arquivos desenvolvidos e a base utilizada para apresentação acadêmica.

## Objetivos

- Aplicar conceitos fundamentais de lógica de programação.
- Desenvolver uma solução prática em grupo.
- Organizar o projeto em um repositório versionado.
- Documentar o funcionamento e a proposta da solução.
- Exercitar comunicação, divisão de tarefas e apresentação técnica.

## Funcionalidades

- Estrutura de projeto acadêmico para apresentação do CodeQuest.
- Arquivos de código e materiais relacionados ao desenvolvimento.
- Organização do projeto para consulta, evolução e portfólio.

## Tecnologias utilizadas

As tecnologias podem variar conforme os arquivos do repositório, mas o projeto foi construído com foco em fundamentos de programação e desenvolvimento acadêmico.

## Como executar

```bash
# Clone o repositório
git clone https://github.com/pgmaximo/S1_ProjetoIntegrador_CodeQuest.git

# Acesse a pasta do projeto
cd S1_ProjetoIntegrador_CodeQuest
```

Depois disso, abra os arquivos no editor de sua preferência e execute conforme a tecnologia principal presente no projeto.
